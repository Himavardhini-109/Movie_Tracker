# Movie_Tracker
📌 Project Overview
This project focuses on analyzing movie data using Python, SQL, and Tableau to extract meaningful insights and visualize trends.
The final output is an interactive dashboard that helps understand movie ratings, genre performance, and trends over time.

---

🚀 Features

- 📊 Top 10 Movies by Rating
- 🎭 Top Genres by Average Rating
- 📈 Movie Trends Over Years
- 🎯 Interactive Filters (Genre, Rating, etc.)
- 🔍 Highlight & Filter Actions for better analysis

---

🛠️ Tools & Technologies Used

- Python → Data cleaning & preprocessing
- SQL → Data querying & transformation
- Tableau → Data visualization & dashboard creation
